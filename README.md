# Amazon Products Exporter

Extensión de Chrome para exportar resultados de búsqueda de Amazon a CSV o JSON.

## Instalación

1. Descarga o clona este repositorio
2. Abre Chrome y ve a `chrome://extensions/`
3. Activa el **Modo desarrollador** (esquina superior derecha)
4. Haz clic en **Cargar descomprimida** y selecciona la carpeta del proyecto

## Uso

1. Ve a cualquier página de resultados de Amazon
2. Haz clic en el ícono de la extensión
3. Elige el formato (CSV/JSON) y exporta
